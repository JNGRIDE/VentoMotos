# **App Name**: MotoSales CRM

## Core Features:

- Multi-User Authentication: Secure authentication with role-based access control (Manager, Salesperson) via Firebase Auth.
- Sprint Management: Configuration of monthly sales and credit goals at the start of each month.
- Performance Dashboard: Interactive charts displaying sales progress, commission eligibility (1.9% at 80% target), and Vento credit bonus eligibility ($200 from 5th credit).
- Sales Recording: Detailed form for recording sales with payment method (Cash, Financing), down payment, and monthly installments.
- Prospect CRM: Sales funnel (Potential -> Appointment -> Credit -> Closed) with source tracking (Organic/Advertising).
- PDF Report Generation: Generate and export PDF sales reports for management review.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF) for a modern and trustworthy feel, referencing the brand color of many motorcycle companies.
- Background color: Light sky blue (#B0E2FF) for a soft and unobtrusive background.
- Accent color: Electric violet (#8F00FF) to highlight important interactions, contrasting effectively with the light blue background.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- Use sharp, modern icons to represent sales metrics, user roles, and sales funnel stages.
- Responsive design for mobile and desktop, ensuring consistent experience across devices.
- Subtle transitions and animations for loading data and updating dashboards.
